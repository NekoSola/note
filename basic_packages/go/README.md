# 简介
标准库 go 主要提供的语言层面的，在一般项目中很少用到. 主要用于代码质量监测如 golint 等类型的项目.


## 模块
+ ast
+ build
+ constant
+ doc
+ format
+ importer
+ parser
+ printer
+ scanner
+ token
+ types

## 开源项目
使用标准库 go 的一些开源项目有:
+ [golint](github.com/golang/lint)
+ [go revive](github.com/mgechev/revive)
